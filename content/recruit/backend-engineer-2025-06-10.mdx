# バックエンドエンジニア募集

スタラップでは、スケーラブルで信頼性の高いバックエンドシステムの設計・開発を担当していただけるエンジニアを募集しています。

## 募集要項

### 職種
バックエンドエンジニア

### 雇用形態
正社員

### 仕事内容

- **API設計・開発**: RESTful APIやGraphQL APIの設計・実装
- **データベース設計**: スケーラブルなデータベース設計と最適化
- **インフラ構築**: クラウドインフラの設計・構築・運用
- **パフォーマンス改善**: システムのボトルネック解析と改善

### 必須スキル

- Go、Python、Node.js等のバックエンド言語での実務経験（3年以上）
- RDBMSの設計・運用経験
- クラウドサービス（AWS、GCP等）の使用経験
- RESTful APIの設計・実装経験

### 歓迎スキル

- マイクロサービスアーキテクチャの経験
- Kubernetes、Dockerの運用経験
- CI/CDパイプラインの構築経験
- セキュリティに関する知識

## 開発環境

- **言語**: Go, Python, TypeScript
- **フレームワーク**: Gin, FastAPI, Express
- **データベース**: PostgreSQL, Redis, MongoDB
- **インフラ**: AWS, Kubernetes, Docker
- **CI/CD**: GitHub Actions, CircleCI
