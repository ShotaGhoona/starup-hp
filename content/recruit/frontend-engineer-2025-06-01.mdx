# フロントエンドエンジニア募集

スタラップでは、最新のフロントエンド技術を活用し、ユーザー体験を革新するプロダクト開発に携わっていただけるエンジニアを募集しています。

## 募集要項

### 職種
フロントエンドエンジニア

### 雇用形態
正社員

### 仕事内容

- **プロダクト開発**: React/Next.jsを使用した自社SaaSプロダクトの開発
- **UI/UX改善**: デザイナーと協力し、最高のユーザー体験を実現
- **技術選定**: 新しい技術の評価・導入を主導
- **コードレビュー**: チームメンバーのコードレビューとメンタリング

### 必須スキル

- React/Next.jsを使用した実務経験（2年以上）
- TypeScriptの実務経験
- REST APIやGraphQLの使用経験
- Git/GitHubを使った開発経験

### 歓迎スキル

- モダンなCSSフレームワーク（Tailwind CSS等）の経験
- テスト駆動開発の経験
- パフォーマンスチューニングの経験
- UI/UXデザインへの理解

## 開発環境

- **言語**: TypeScript, JavaScript
- **フレームワーク**: Next.js, React
- **スタイリング**: Tailwind CSS, CSS Modules
- **状態管理**: React Context, Zustand
- **テスト**: Jest, React Testing Library
- **インフラ**: Vercel, AWS

東京オフィス（渋谷）/ リモート可

- リモートワーク手当

- 資格取得支援
